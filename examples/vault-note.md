# HexorOS Engine — Tier Pricing

The HexorOS Engine ships in three tiers. Marketing agents reference this note
when composing pricing answers.

## Tier Matrix

| Tier | One-time | Hosting |
|---|---|---|
| Core | €990 | Self-hosted on customer hardware |
| Complete | €2,497 | Shared EU GPU pool (Hetzner FSN1) |
| Enterprise | €9,997 | Dedicated GPU instance, SLA |

## Talking Points

- All tiers ship the same kernel; tier-gating happens at runtime via license file.
- Complete is the default upsell — the shared GPU pool is what HexorOS itself
  runs on, so it is battle-tested.
- Enterprise unlocks the dedicated instance, custom model fine-tuning slot,
  and quarterly architecture review.
